## **About dataset**           
This dataset provides sales information for Balaji Fast Food Restaurant, including various food items. The dataset includes the following fields:
- The data is a CSV file with 10 columns and 1001 rows.
    * Date
    * Item Name
    * Item Type
    * Item Price
    * Quantity
    * Transaction Amount
    * Transaction Type
    * Received By
    * Time of Sale
